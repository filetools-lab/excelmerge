    FAQ

    Common questions

    Answers for privacy, file formats, merge behavior, and mismatch checks.

    Can I merge XLSX files?

    Yes. This free browser based Excel merge tool supports XLSX, XLS and CSV files.

    Are files uploaded?

    No. All processing happens locally in your browser, so you can merge Excel files without uploading sensitive data.

    Does this work offline?

    Yes. After loading the page you can continue working offline in the same browser tab.

    Is my data secure?

    Yes. Your files never leave your computer, which makes this useful for financial worksheets, sales reports and private business data.

    Is there a file size limit?

    There is no server file size limit. The practical limit is your browser memory and device performance.
