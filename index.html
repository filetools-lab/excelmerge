    FAQ

    Common questions

    Answers for privacy, file formats, merge behavior, and mismatch checks.

    Are my Excel files uploaded to a server?

    No. Files are read and merged locally in your browser. Workbook contents are not uploaded to an application server.

    Which file formats are supported?

    You can merge .xlsx, .xls, and .csv files.

    Can I merge multiple worksheets?

    Yes. You can combine all rows into one worksheet or keep each source worksheet as a separate sheet in one workbook.

    What happens if headers are different?

    The tool shows a warning with the file and sheet location before merging. You can return to check files or continue merging.

    Review needed

    Some tables do not match

    The files can still be merged, but these differences may affect the output. Please review them first.
